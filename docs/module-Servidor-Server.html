Class: Server

Servidor~Server()

Clase para crear un servidor. Esta clase implementa un servidor para conectarse a la base de datos, procesar las solicitudes que se hagan a sus rutas y posteriormente se gestionen las consultas a la base de datos. El servidor está implementado con el Framework Express y su trabajo principal es colocarse a la escucha en un puerto declarado y poder empezar a recibir solicitudes desde el Frontend.

Constructor

new Server()

Constructor de la clase Servidor. El constructor inicializa el servidor con Express y declara los atributos necesarios para poder poner en funcionamiento el servidor. (Paths, puerto, middlewares y rutas).
